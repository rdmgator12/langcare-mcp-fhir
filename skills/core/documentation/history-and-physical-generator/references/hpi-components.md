# HPI Components, ROS Checklist, and Physical Exam Documentation

## History of Present Illness (HPI)

### OLDCARTS Framework

Use OLDCARTS to structure every HPI systematically. Each element must be addressed or explicitly noted as not applicable.

| Component | Description | Documentation Prompts |
|-----------|------------|----------------------|
| **O**nset | When did the symptom/problem start? | Sudden vs gradual, exact date/time if known, precipitating event |
| **L**ocation | Where is the symptom? | Anatomic location, laterality, radiation pattern |
| **D**uration | How long does it last? | Constant vs intermittent, episode length, total duration of illness |
| **C**haracter | What does it feel like? | Quality descriptors (sharp, dull, burning, pressure, cramping, aching, throbbing) |
| **A**ggravating | What makes it worse? | Activity, position, food, time of day, stress, specific triggers |
| **R**elieving | What makes it better? | Rest, medication, position change, heat/cold, specific interventions tried |
| **T**emporal | Pattern over time? | Morning vs evening, relationship to meals, worsening/improving trend, frequency |
| **S**everity | How bad is it? | Pain scale 0-10, functional impact (ability to work, sleep, ADLs) |

### Additional HPI Elements

Beyond OLDCARTS, a comprehensive HPI should address:

| Element | Description |
|---------|------------|
| Associated symptoms | Related symptoms across organ systems |
| Prior episodes | Has this happened before? When? How was it treated? |
| Prior workup | Previous testing for this complaint (labs, imaging, specialist evaluation) |
| Impact on function | Effect on work, sleep, appetite, mobility, ADLs |
| Patient's concern | What is the patient worried about? What prompted this visit now? |
| Relevant negatives | Important symptoms that are NOT present (helps narrow differential) |

### HPI Documentation Standards

**Problem-focused HPI (1-3 elements):** Brief, single-problem visit
- Example: "Patient reports 2 days of sore throat (onset, duration, location). No fever."

**Extended HPI (4+ elements OR status of 3+ chronic conditions):** Complex visit
- Example: "62-year-old male with 3-hour history (onset) of substernal (location) crushing chest pain (character) radiating to the left arm, rated 8/10 (severity), constant (duration), worse with exertion (aggravating), partially relieved by rest and nitroglycerin (relieving), associated with diaphoresis and dyspnea."

**Chronic Condition Status Update:**
For each chronic condition, document:
- Current symptom status
- Compliance with treatment
- Side effects of therapy
- Relevant interval changes since last visit
- Current functional status related to condition

## Review of Systems (ROS) -- Complete 14-System Checklist

### Documentation Format
For each system: document pertinent positives AND pertinent negatives. "All other systems reviewed and negative" is acceptable only if you have directly asked or reviewed each system.

## Physical Examination Documentation by System

### General / Constitutional
```
General: [Age]-year-old [gender] appearing [stated age / younger / older].
[Well-developed / thin / obese], [well-nourished / malnourished].
[No acute distress / mild distress / moderate distress / severe distress].
Alert, oriented x [3 (person, place, time) / 4 (+situation)].
```

### HEENT
```
Head: Normocephalic, atraumatic. [Scalp normal / lesions noted].
Eyes: PERRL (pupils equal, round, reactive to light). EOMI (extraocular movements intact).
  Conjunctivae clear. Sclerae anicteric. [Fundoscopic exam findings if performed].
Ears: External ears normal. TMs [clear / dull / bulging / retracted] bilaterally.
  Canals clear. Hearing [intact to conversation / decreased].
Nose: [Patent / congested]. No septal deviation. [Mucosa normal / erythematous / pale].
  [No discharge / clear discharge / purulent discharge].
Throat: Oropharynx [clear / erythematous]. [Tonsils normal / enlarged (grade 1-4)].
  [No exudates / exudates present]. Uvula midline. Moist mucous membranes.
```

### Neck
```
Neck: Supple. Full ROM. No cervical lymphadenopathy.
  [No JVD / JVD to X cm above sternal angle].
  Trachea midline. Thyroid [normal / enlarged / nodular / not palpable].
  [No carotid bruits / carotid bruit noted on [side]].
```

### Cardiovascular
```
CV: [Regular rate and rhythm / irregularly irregular / regularly irregular].
  S1, S2 [normal / distant / loud]. [No murmurs / systolic murmur grade [I-VI/VI] at [location]].
  [No gallops / S3 present / S4 present]. [No rubs / friction rub noted].
  PMI [normal / displaced laterally].
  Peripheral pulses [2+ and symmetric / diminished / absent] [specify location].
  [No peripheral edema / [1+/2+/3+/4+] pitting edema [bilateral / unilateral] [distribution]].
  [No cyanosis / cyanosis noted]. Capillary refill [<2 sec / delayed].
```

### Pulmonary
```
Lungs: [Comfortable on room air / in respiratory distress / labored breathing].
  [No accessory muscle use / using accessory muscles].
  [Clear to auscultation bilaterally / decreased breath sounds at [location]].
  [No wheezes, rhonchi, or rales / wheezing [diffuse/focal] / crackles at [bases/fields]].
  [Symmetric chest expansion / asymmetric].
  [No dullness to percussion / dull to percussion at [location]].
  [Tactile fremitus normal / increased / decreased].
```

### Abdomen
```
Abdomen: [Soft / firm / rigid / distended]. [Non-tender / tender in [quadrant/region]].
  [No guarding / guarding present]. [No rebound / rebound tenderness].
  Bowel sounds [normoactive / hypoactive / hyperactive / absent] in all quadrants.
  [No hepatosplenomegaly / liver palpable X cm below costal margin / spleen palpable].
  [No masses / mass palpated in [location]].
  [No CVA tenderness / CVA tenderness on [side]].
  [No hernias / [type] hernia at [location]].
```

### Musculoskeletal
```
MSK: [Spine/extremities to document]:
  [Joint]: [No swelling, erythema, warmth / swollen, erythematous, warm].
  ROM: [Full / limited in [direction]]. [No tenderness / tenderness at [location]].
  Strength: [5/5 / X/5] [muscle group] [bilaterally / asymmetric].
  Gait: [Normal / antalgic / ataxic / wide-based / shuffling].
  [No deformity / deformity noted].
```

### Neurological
```
Neuro: Mental status: Alert, oriented x [3-4]. [Appropriate affect / flat / labile].
  Cranial nerves II-XII [intact / deficit noted in CN [number]]:
    II: Visual acuity, visual fields
    III, IV, VI: EOM, pupils
    V: Facial sensation, masseter strength
    VII: Facial symmetry
    VIII: Hearing
    IX, X: Palate elevation, gag reflex
    XI: Trapezius, SCM strength
    XII: Tongue midline, no fasciculations
  Motor: [5/5 strength throughout / weakness in [distribution]].
  Sensory: [Intact to light touch, pinprick, proprioception / decreased in [distribution]].
  Reflexes: [2+ and symmetric / hyperreflexic / hyporeflexic / absent] [distribution].
  [Babinski negative / positive bilaterally].
  Coordination: [Finger-to-nose intact / dysmetria]. [Heel-to-shin intact / impaired].
  Gait: [Normal / ataxic / Romberg negative / Romberg positive].
```

```
Psych: Appearance: [Grooming, dress, hygiene].
  Behavior: [Cooperative, agitated, withdrawn, psychomotor retardation].
  Speech: [Normal rate/rhythm/volume / pressured / slow / loud / soft].
  Mood: [Patient's stated mood in quotes].
  Affect: [Congruent/incongruent, full range/restricted/flat/labile].
  Thought process: [Linear, goal-directed / tangential / circumstantial / loose].
  Thought content: [No SI/HI / SI present without plan / with plan].
    [No delusions / paranoid ideation / grandiose].
    [No hallucinations / auditory hallucinations / visual hallucinations].

```

### Skin
```

  [No rashes or lesions / rash described: morphology, distribution, color, size].
```
