# Procedure Documentation by Procedure Type

## Required Elements for All Bedside Procedures

Every procedure note must contain these universal elements regardless of procedure type:

| Element | Description |
|---------|------------|
| Date, time, location | When and where the procedure was performed |
| Operator and supervisor | Names, credentials, service |
| Indication | Why the procedure was necessary |
| Consent | Informed consent status (verbal, written, emergency) |
| Time-out | Universal protocol verification documented |
| Allergies | Patient allergies reviewed |
| Anesthesia/sedation | Type and agents used |
| Technique | Step-by-step description |
| Findings | What was found during the procedure |
| Specimens | What was collected and where sent |
| Complications | Any complications or "none" |
| EBL | Estimated blood loss |
| Post-procedure | Patient tolerance, orders, follow-up imaging |

## Central Venous Catheter Insertion

### Pre-Procedure Requirements

| Item | Details |
|------|---------|
| Indications | Vasopressor administration, TPN, poor IV access, hemodialysis, CVP monitoring, frequent blood draws |
| Labs required | INR (< 1.5 preferred), Platelets (> 50k), aPTT |
| Contraindications (relative) | Coagulopathy, overlying infection, anatomic distortion, prior radiation to site |
| Equipment | CVC kit (triple-lumen 7Fr or quad-lumen), ultrasound, sterile gown/gloves/drapes, chlorhexidine, lidocaine, suture kit |

### Procedure-Specific Documentation

```
CENTRAL VENOUS CATHETER INSERTION

Site: [Right/Left] [Internal jugular / Subclavian / Femoral]
Catheter: [Type] [Size Fr] [Number of lumens] [Brand if relevant]
Approach: [Anterior / Posterior / Lateral] [Seldinger technique]

Ultrasound guidance: [Yes / No]
  Pre-procedure: Vessel identified, compressible, patent
  Real-time: Needle visualized entering vessel
  Post-procedure: Wire confirmed in vessel

Sterile technique: Full barrier precautions (cap, mask, sterile gown, sterile gloves, full body drape)
Skin prep: [Chlorhexidine / Betadine] -- allowed to dry [time]
Local anesthesia: Lidocaine [concentration]% [volume] mL

Technique:
  - [Site prepped and draped in sterile fashion]
  - [Ultrasound used to identify vessel / landmarks identified]
  - [Local anesthesia infiltrated]
  - [Vessel accessed with [gauge] needle under ultrasound guidance on attempt #[n]]
  - [Dark non-pulsatile blood returned / Bright pulsatile blood -- arterial, needle removed]
  - [Guidewire advanced without resistance]
  - [Wire position confirmed by ultrasound in [IJ/vessel]]
  - [Nick incision made with #11 blade]
  - [Dilator passed over wire]
  - [Catheter advanced over wire to [depth] cm]
  - [Wire removed, all ports aspirated and flushed with saline]
  - [Catheter secured with [suture / StatLock]]
  - [Sterile dressing applied]

Catheter depth: [cm] at skin
Number of attempts: [n]
Blood return from all ports: [Yes / No -- specify]
Complications: [None / Arterial puncture / Pneumothorax / Hematoma / Arrhythmia]

Post-procedure:
  - CXR ordered for line position confirmation and pneumothorax evaluation
  - CXR result: [Tip at cavoatrial junction / tip in SVC / tip too high or low / no pneumothorax]

```

## Thoracentesis

### Procedure-Specific Documentation

```
THORACENTESIS

Indication: [Diagnostic (new effusion) / Therapeutic (large effusion, dyspnea)]
Pre-procedure imaging: [CXR / CT / Ultrasound] showing [side] pleural effusion, estimated [size]

Site: [Right / Left] posterior thorax, [intercostal space] midscapular/posterior axillary line
Ultrasound guidance: [Yes -- effusion confirmed, depth [cm], no intervening lung]
Sterile technique: [Full barrier / Modified sterile]
Skin prep: [Chlorhexidine / Betadine]
Local anesthesia: Lidocaine [concentration]% [volume] mL

Technique:
  - [Patient positioned sitting upright, leaning forward on bedside table]
  - [Ultrasound used to confirm effusion and mark site]
  - [Site prepped and draped]
  - [Local anesthesia infiltrated -- skin, rib periosteum, parietal pleura]
  - [Thoracentesis needle/catheter advanced over superior aspect of rib]
  - [Pleural fluid obtained on attempt #[n]]
  - [Fluid appearance: clear yellow / turbid / bloody / milky]

Volume removed: [total] mL
  - Large volume: stopped at [volume] or when [patient coughed / felt chest tightness / no more fluid]
  - Maximum recommended single session: 1500 mL (to prevent re-expansion pulmonary edema)

Fluid appearance: [Serous / Serosanguinous / Hemorrhagic / Purulent / Chylous]

Specimens sent:
  - Cell count with differential
  - Protein, albumin, LDH
  - Glucose
  - pH (in heparinized syringe, on ice -- send as blood gas)
  - Gram stain and culture
  - [Cytology / AFB culture and smear / Triglycerides / Amylase / Adenosine deaminase]

Light's criteria labs to determine exudate vs transudate:
  - Pleural protein / Serum protein ratio (> 0.5 = exudate)
  - Pleural LDH / Serum LDH ratio (> 0.6 = exudate)
  - Pleural LDH > 2/3 upper limit of normal serum LDH (= exudate)

Serum protein and LDH: [Ordered simultaneously]

Number of attempts: [n]
Complications: [None / Pneumothorax / Cough / Re-expansion symptoms / Vasovagal / Bleeding]

Post-procedure:
  - Post-thoracentesis CXR: [Ordered / Result: no pneumothorax / small pneumothorax]
  - Symptom improvement: [Yes -- dyspnea improved / No change]
  - Monitor for: dyspnea, chest pain, cough, hypotension
```

```
Indication: [Pneumothorax / Hemothorax / Empyema / Malignant effusion / Post-operative]
Urgency: [Emergent (tension pneumothorax) / Urgent / Elective]

Site: [Right / Left] [4th-5th intercostal space, anterior-midaxillary line (safe triangle)]
Tube: [Size Fr] [Type: surgical / pigtail catheter]
Sterile technique: Full barrier precautions
Skin prep: [Chlorhexidine / Betadine]
Local anesthesia: Lidocaine [concentration]% [volume] mL (max dose documented)

Technique (surgical):
  - [Patient positioned semi-upright, arm abducted above head]
  - [Site prepped and draped]
  - [Local anesthesia infiltrated to skin, subcutaneous tissue, periosteum, pleura]
  - [2-3 cm incision made along intercostal space]
  - [Blunt dissection through subcutaneous tissue with Kelly clamp]
  - [Pleura punctured with clamp -- rush of air/fluid confirmed entry]
  - [Digital sweep to confirm no adhesions / no lung adherent to chest wall]
  - [Chest tube advanced [superiorly for pneumothorax / posteriorly and inferiorly for fluid]]
  - [Tube positioned to [depth] cm at skin]
  - [Connected to [Pleurevac / Atrium] drainage system at -20 cm H2O suction]
  - [Tube secured with [suture technique] and covered with occlusive dressing]

Immediate output: [volume] mL [character: air / serous / blood / purulent]
Tidaling: [Present / Absent]
Air leak: [None / Present on cough / Present at rest]
Respiratory variation: [Present -- tube functioning]

Complications: [None / Subcutaneous placement / Lung laceration / Bleeding / Intercostal artery injury]

Post-procedure:
  - CXR: [Lung re-expanded / residual pneumothorax / residual effusion / tube position satisfactory]
  - Drainage system: Functioning, suction confirmed
  - Monitor: Output, air leak, respiratory status
```
