# SOAP Note Documentation Standards

## SOAP Note Structure

### Subjective (S)

The subjective section captures patient-reported information.

#### Chief Complaint (CC)
- Document in the patient's own words when possible
- Single concise statement: symptom + duration
- Examples: "Chest pain for 2 hours", "Cough for 3 weeks", "Follow-up for diabetes"

#### History of Present Illness (HPI)
Use the OLDCARTS mnemonic:
- **O**nset: When did symptoms begin? Sudden or gradual?
- **L**ocation: Where is the symptom? Does it radiate?
- **D**uration: How long does each episode last? Constant or intermittent?
- **C**haracterization: Quality of symptom (sharp, dull, burning, pressure)
- **A**ggravating factors: What makes it worse?
- **R**elieving factors: What makes it better? Medications tried?
- **T**emporal patterns: Time of day? Frequency? Progression?
- **S**everity: Pain scale 0-10, functional impact

#### Review of Systems (ROS)
14-system checklist. Document pertinent positives AND negatives.

| System | Key Elements |
|--------|-------------|
| Constitutional | Fever, chills, fatigue, weight change, night sweats |
| Eyes | Vision changes, pain, redness, discharge |
| ENT | Hearing loss, tinnitus, sore throat, congestion, epistaxis |
| Cardiovascular | Chest pain, palpitations, edema, orthopnea, PND, claudication |
| Respiratory | Dyspnea, cough, wheezing, hemoptysis, sputum production |
| GI | Nausea, vomiting, diarrhea, constipation, abdominal pain, melena, hematochezia |
| GU | Dysuria, frequency, urgency, hematuria, incontinence, discharge |
| Musculoskeletal | Joint pain, swelling, stiffness, back pain, weakness |
| Integumentary | Rashes, lesions, pruritus, wound changes |
| Neurological | Headache, dizziness, syncope, numbness, tingling, weakness, seizures |
| Psychiatric | Depression, anxiety, insomnia, suicidal ideation, hallucinations |
| Endocrine | Polyuria, polydipsia, heat/cold intolerance, hair changes |
| Hematologic | Easy bruising, bleeding, lymphadenopathy, prior DVT/PE |
| Allergic/Immunologic | Seasonal allergies, frequent infections, autoimmune history |

**ROS Documentation Requirements by Visit Level:**
- Problem-focused: Pertinent system only
- Expanded problem-focused: 2-9 systems
- Detailed: 10+ systems
- Comprehensive: All 14 systems (complete ROS)

### Objective (O)

#### Vital Signs
Document all available:
- Temperature (oral, tympanic, rectal, axillary -- note method)
- Heart rate (regular/irregular)
- Blood pressure (position, which arm, cuff size if relevant)
- Respiratory rate
- Oxygen saturation (with FiO2 or delivery device)
- Weight (note if patient-reported vs measured)
- Height
- BMI (calculated)
- Pain score

#### Physical Examination Templates by Specialty

**Primary Care / Internal Medicine:**
| System | Components |
|--------|-----------|
| General | Appearance, distress level, orientation, nutritional status |
| HEENT | Head normocephalic/atraumatic, PERRL, EOMI, TMs clear, oropharynx clear, no JVD |
| Neck | Supple, no lymphadenopathy, thyroid normal |
| Cardiovascular | Rate/rhythm, murmurs/gallops/rubs, peripheral pulses, edema |
| Pulmonary | Effort, breath sounds bilateral, adventitious sounds |
| Abdomen | Soft, bowel sounds, tenderness, organomegaly, masses |
| Extremities | Edema, cyanosis, clubbing, ROM |
| Neurological | CN II-XII, strength, sensation, reflexes, gait |
| Skin | Lesions, rashes, wounds, turgor |
| Psychiatric | Mood, affect, thought process, judgment |

**Emergency Department:**
All of the above plus:
- Trauma primary survey (ABCDE) when applicable
- Focused exam based on chief complaint
- Reassessment documentation with timestamps

**Pediatric:**
Additional elements:
- Growth parameters (percentiles)
- Developmental milestones
- Fontanelle status (infants)
- Tanner staging (adolescents)

#### Laboratory and Imaging
- Report significant results with reference ranges
- Flag abnormal and critical values
- Note pending results
- Compare to prior values when relevant

### Assessment (A)

#### Problem List Format
Number each problem. Include:
1. Diagnosis or differential (most likely first)
2. ICD-10 code when known
3. Clinical reasoning supporting the assessment
4. Acuity: acute, chronic, acute-on-chronic

**Differential Diagnosis Documentation:**
For undifferentiated complaints, list top 3-5 differentials with supporting/refuting evidence for each.

### Plan (P)

#### Per-Problem Plan Elements
For each assessment item:
- **Diagnostics**: Labs ordered, imaging ordered, referrals
- **Therapeutics**: Medications (new, changed, continued), procedures
- **Patient education**: Counseling provided, resources given
- **Disposition**: Follow-up timing, return precautions
- **Coordination**: Referrals, communication with other providers

#### Standard Plan Elements
- Activity level
- Diet
- Follow-up appointment
- Return-to-work/school status
- Preventive care addressed
- Shared decision-making documented

## Documentation Requirements by Visit Type

### New Patient Visit
- Comprehensive history (CC, HPI, ROS 10+ systems, PFSH complete)
- Comprehensive exam (8+ organ systems)
- MDM or time-based billing

### Established Patient Visit
- Based on 2021 E&M guidelines: medical decision-making OR total time determines level
- No minimum history or exam requirements
- Document what is clinically relevant

### Preventive / Wellness Visit
- Age-appropriate screening review
- Risk factor assessment
- Immunization status
- Counseling and anticipatory guidance
- Separate from problem-oriented E&M when both occur same day

### Urgent / Same-Day Visit
- Focused on acute complaint
- Document medical necessity for same-day visit
- Time-sensitive elements (onset, progression)

### Telehealth Visit
- Document modality (video, audio-only, remote monitoring)
- Note patient location and provider location
- Technology limitations affecting exam
- Document elements that could not be assessed remotely

## Time-Based Documentation

When billing on time (per 2021 guidelines for office visits):
- Document total time on date of encounter
- Include: reviewing records, ordering tests, communicating with other providers, documenting, care coordination
- Time does not need to be face-to-face for office visits
- Separately billable: prolonged services (add-on codes)

| CPT | Time Range | Level |
|-----|-----------|-------|
| 99212/99202 | 10-19 min | Level 2 |
| 99213/99203 | 20-29 min | Level 3 |
| 99214/99204 | 30-39 min | Level 4 |
| 99215/99205 | 40-54 min | Level 5 |
| +99417 | Each additional 15 min | Prolonged |
