# Morphine Milligram Equivalent (MME) Conversion Table

Source: CDC Clinical Practice Guideline for Prescribing Opioids (2022), CMS MME Calculation methodology. Based on equianalgesic dose ratios relative to oral morphine.

## Conversion Factors

The MME conversion factor converts the dose of a given opioid to its morphine equivalent.

```
Daily MME = Daily dose of opioid (mg) x MME conversion factor
```

### Fentanyl Conversions

| Formulation | Conversion | Notes |
|------------|-----------|-------|
| Transdermal (patch) mcg/hr | mcg/hr x 2.4 = daily MME | 25mcg/hr patch = 60 MME/day |
| Transmucosal (lozenge, film) mcg | Do NOT convert to MME. These are for breakthrough pain in opioid-tolerant patients only. | Not applicable for total daily dose calculation |
| Parenteral mcg | mcg/day x 0.24 = daily MME | Used for IV/IM fentanyl |
| Nasal spray mcg | Breakthrough dosing, do not add to baseline MME | Assess separately |

### Buprenorphine Conversions

| Formulation | MME Factor | Notes |
|------------|-----------|-------|
| Transdermal patch (mcg/hr) | mcg/hr x 12.6 | For chronic pain (Butrans). 10mcg/hr = 126 MME |
| Sublingual/buccal for pain (mg) | 30.0 | For chronic pain (Belbuca) |
| Sublingual for OUD (Suboxone, Subutex) | NOT included in MME | Medication for opioid use disorder. Do not include in opioid risk MME calculation. |

## Methadone Conversion (Variable Factor)

Methadone has a unique, non-linear conversion factor that increases with dose. This is due to its long and variable half-life (8-59 hours) and NMDA receptor activity.

| Total Daily Oral Methadone Dose | MME Conversion Factor |
|--------------------------------|----------------------|
| 1-20 mg/day | 4.0 |
| 21-40 mg/day | 8.0 |
| 41-60 mg/day | 10.0 |
| >= 61 mg/day | 12.0 |

|---------------------|--------|-----------|

| 10mg | 4.0 | 40 |
| 20mg | 4.0 | 80 |
| 30mg | 8.0 | 240 |
| 40mg | 8.0 | 320 |
| 50mg | 10.0 | 500 |
| 60mg | 10.0 | 600 |
| 80mg | 12.0 | 960 |
| 100mg | 12.0 | 1200 |

**Methadone for OUD**: When methadone is prescribed specifically for opioid use disorder maintenance (dispensed through an OTP), it is generally NOT included in the MME risk calculation for pain prescribing thresholds. However, its presence should still be noted for overdose risk and CNS depression assessment.

## Extended-Release / Long-Acting (ER/LA) Formulations

ER/LA formulations provide continuous opioid exposure. Calculate daily MME based on the labeled daily dose.

| Product | Typical Dosing | Daily Dose Calculation |
|---------|---------------|----------------------|
| Morphine ER (MS Contin) | q12h | Single dose x 2 = daily dose. MME factor: 1.0 |
| Oxycodone ER (OxyContin) | q12h | Single dose x 2 = daily dose. MME factor: 1.5 |
| Hydromorphone ER (Exalgo) | q24h | Labeled dose = daily dose. MME factor: 5.0 |
| Oxymorphone ER (Opana ER) | q12h | Single dose x 2 = daily dose. MME factor: 3.0 |
| Tapentadol ER (Nucynta ER) | q12h | Single dose x 2 = daily dose. MME factor: 0.4 |
| Tramadol ER | q24h | Labeled dose = daily dose. MME factor: 0.2 |
| Fentanyl transdermal | q72h | mcg/hr x 2.4 = daily MME |
| Methadone | Variable (q8-12h) | Sum all daily doses, apply variable factor |
| Buprenorphine transdermal (Butrans) | q7d | mcg/hr x 12.6 = daily MME |

## Calculation Examples

### Example 1: Single Opioid
Patient on oxycodone 10mg every 6 hours:
- Daily dose: 10mg x 4 = 40mg/day
- MME: 40mg x 1.5 = **60 MME/day** [CAUTION threshold exceeded]

### Example 2: Multiple Opioids
Patient on fentanyl 50mcg/hr patch + oxycodone 15mg q6h PRN (takes 3 per day):
- Fentanyl: 50 x 2.4 = 120 MME
- Oxycodone: 15mg x 3 = 45mg/day x 1.5 = 67.5 MME
- Total: **187.5 MME/day** [HIGH RISK - far exceeds 90 MME]

### Example 3: Codeine Combination Product
Patient on Tylenol #3 (codeine 30mg/acetaminophen 300mg), 2 tablets q6h:
- Codeine daily: 30mg x 2 tablets x 4 doses = 240mg/day
- MME: 240mg x 0.15 = **36 MME/day** [Below caution threshold]

### Example 4: Methadone (Variable Conversion)
Patient on methadone 40mg BID for chronic pain:
- Total daily methadone: 80mg/day
- Factor for >=61mg/day: 12.0
- MME: 80 x 12.0 = **960 MME/day** [CRITICAL]

### Example 5: Tramadol
Patient on tramadol 50mg QID:
- Daily dose: 50mg x 4 = 200mg/day
- MME: 200mg x 0.2 = **40 MME/day** [Below caution threshold]

## Important Notes

### Medications NOT Included in MME Calculations
- Buprenorphine for opioid use disorder (Suboxone, Subutex, Sublocade)
- Methadone dispensed through OTPs for OUD maintenance (note separately)
- Naloxone (opioid antagonist)
- Naltrexone (opioid antagonist)
- Loperamide (peripheral mu-opioid agonist, minimal CNS penetration at therapeutic doses)
- Diphenoxylate/atropine (Lomotil) at standard doses

### Tolerance Considerations
- Patients on chronic opioid therapy develop tolerance
- MME thresholds still apply for risk assessment regardless of tolerance
- Cross-tolerance between opioids is incomplete (reduce new opioid dose by 25-50% when rotating)
- Tolerance loss occurs rapidly (within 3-7 days of discontinuation). Patients restarting after a gap are at high overdose risk at their previous dose.

### Maximum Recommended Daily Doses
| Opioid | Maximum Recommended Dose | Reason |
|--------|------------------------|--------|
| Codeine | 360mg/day | Ceiling effect, GI side effects |
| Tramadol IR | 400mg/day | Seizure risk |
| Tramadol ER | 300mg/day | Seizure risk |
| Tapentadol IR | 700mg/day (600mg on first day) | Seizure risk |
| Tapentadol ER | 500mg/day | Seizure risk |
| Acetaminophen component | 3000-4000mg/day total from all sources | Hepatotoxicity |
