# Critical Value Thresholds

Critical (panic) values per CAP (College of American Pathologists) guidelines. These values require immediate clinician notification and action. Thresholds may vary by institution; values below represent widely accepted consensus thresholds.

## Notification Requirements

Per CAP/CLIA regulations:
- Critical values must be reported to a responsible licensed caregiver within **60 minutes** of result verification
- Read-back verification is required
- Documentation of notification (who was notified, time, read-back confirmation) is mandatory
- Repeat critical values within the same episode of care may follow institutional policy for reduced notification frequency

## Immediate Action by Critical Value Category

### Critical Potassium (> 6.5 mEq/L)
1. Verify specimen (hemolysis is most common cause of falsely elevated K+)
2. Obtain 12-lead ECG immediately
3. If ECG changes (peaked T waves, widened QRS): calcium gluconate 10% IV, 10mL over 2-3 minutes
4. Insulin 10 units regular IV + D50 25g IV (shifts K+ intracellularly)
5. Albuterol nebulizer 10-20mg (additional intracellular shift)
6. Kayexalate 30g PO or sodium zirconium cyclosilicate for removal
7. Consider emergent dialysis if refractory

### Critical Glucose (< 40 mg/dL)
1. If conscious: 15-20g fast-acting oral glucose, recheck in 15 minutes
2. If altered mental status or unable to take PO: D50 25g (50mL) IV push
3. If no IV access: glucagon 1mg IM
4. Identify and treat cause (insulin dose, missed meal, renal failure, sepsis, adrenal insufficiency)
5. Recheck glucose every 15 minutes until stable > 70 mg/dL

### Critical INR (> 5.0)
1. Assess for active bleeding
2. INR 5.0-9.0, no bleeding: hold warfarin, recheck in 24-48h. Vitamin K 1-2.5mg PO if high bleeding risk.
3. INR > 9.0, no bleeding: hold warfarin + vitamin K 2.5-5.0mg PO
4. Active serious bleeding at any INR: 4-factor PCC (25-50 IU/kg) + vitamin K 10mg IV + fresh frozen plasma if PCC unavailable
5. Life-threatening bleeding: same as above, consider recombinant factor VIIa

### Critical Hemoglobin (< 7.0 g/dL)
1. Type and crossmatch / type and screen if not already done
2. Transfuse pRBCs: threshold < 7 g/dL for most patients, < 8 g/dL for acute coronary syndrome
3. Assess hemodynamic stability (tachycardia, hypotension)
4. Identify source of blood loss (GI, surgical site, retroperitoneal)
5. Consider massive transfusion protocol if ongoing hemorrhage
