# ACOG Prenatal Visit Schedule

Source: American College of Obstetricians and Gynecologists (ACOG) Guidelines for Perinatal Care.

## Standard Visit Frequency

| Gestational Age | Visit Interval | Notes |
|----------------|----------------|-------|
| Initial visit (6-10 weeks) | Confirm pregnancy | Complete history, baseline labs |
| 10-28 weeks | Every 4 weeks | Routine prenatal visits |
| 28-36 weeks | Every 2 weeks | Third trimester monitoring |
| 36 weeks-delivery | Weekly | Late third trimester |
| Postpartum | 3-12 weeks | Comprehensive postpartum visit |

High-risk pregnancies may require more frequent visits.

## Initial Prenatal Visit (First Trimester)

### History
- Complete medical history
- Surgical history
- Obstetric history: GTPAL (Gravida, Term, Preterm, Abortions, Living)
- Menstrual history: LMP, cycle regularity
- Family history: genetic conditions, birth defects, thrombophilia
- Social history: substance use (tobacco, alcohol, drugs), domestic violence screening
- Current medications and supplements
- Allergy review

### GTPAL Documentation

| Component | Definition |
|-----------|-----------|
| G (Gravida) | Total number of pregnancies including current |
| T (Term) | Deliveries at >= 37 weeks |
| P (Preterm) | Deliveries between 20-36 weeks 6 days |
| A (Abortions) | Pregnancy losses before 20 weeks (spontaneous + elective) |
| L (Living) | Number of living children |

### Physical Examination
- Height, weight, BMI (prepregnancy)
- Blood pressure baseline
- Complete physical exam
- Pelvic exam with Pap smear if due
- Uterine sizing

### Optional First Trimester Screening (10-14 weeks)

| Test | Gestational Age | Purpose |
|------|----------------|---------|
| NIPT (cell-free DNA) | >= 10 weeks | Trisomy 21, 18, 13, sex chromosome screening |
| First trimester combined screen | 11-14 weeks | NT ultrasound + PAPP-A + free beta-hCG |
| Chorionic villus sampling (CVS) | 10-13 weeks | Diagnostic (if indicated) |

## Second Trimester Schedule

### Routine Visit Components (Every 4 Weeks, 14-28 Weeks)

Each visit:
- Blood pressure
- Weight
- Urine dipstick (protein, glucose)
- Fundal height (starting ~20 weeks)
- Fetal heart tones (starting ~12 weeks with Doppler)
- Review symptoms: fetal movement (quickening typically 16-20 weeks), contractions, bleeding, fluid leakage

### Second Trimester Screenings and Labs

| Test | Gestational Age | LOINC | Notes |
|------|----------------|-------|-------|
| Quad screen (if NIPT not done) | 15-20 weeks | 49581-2 | AFP, hCG, estriol, inhibin-A |
| Anatomy ultrasound | 18-22 weeks | -- | Fetal anatomy survey, placental location |
| 1-hour glucose challenge (GCT) | 24-28 weeks | 1504-0 | 50g glucose load, threshold 130 or 140 mg/dL |
| 3-hour GTT (if GCT abnormal) | 24-28 weeks | 1518-0, 1530-5, 1547-9, 1507-3 | 100g glucose load |
| Repeat Rh antibody screen | 28 weeks | 890-4 | If Rh-negative |
| Rhogam (RhD immune globulin) | 28 weeks | -- | If Rh-negative and antibody screen negative |
| Repeat CBC | 24-28 weeks | 58410-2 | Reassess for anemia |
| Tdap vaccine | 27-36 weeks | -- | Pertussis prophylaxis |

### 3-Hour GTT Diagnostic Criteria (Carpenter-Coustan)

| Timepoint | LOINC | Threshold |
|-----------|-------|-----------|
| Fasting | 1507-3 | >= 95 mg/dL |
| 1 hour | 1518-0 | >= 180 mg/dL |
| 2 hour | 1530-5 | >= 155 mg/dL |
| 3 hour | 1547-9 | >= 140 mg/dL |

Gestational diabetes diagnosed if >= 2 values met or exceeded.

### 3-Hour GTT Diagnostic Criteria (NDDG Alternative)

| Timepoint | Threshold |
|-----------|-----------|
| Fasting | >= 105 mg/dL |
| 1 hour | >= 190 mg/dL |
| 2 hour | >= 165 mg/dL |
| 3 hour | >= 145 mg/dL |

## Third Trimester Schedule

### Visit Components (Every 2 Weeks 28-36, Weekly 36+)

Each visit:
- Blood pressure (preeclampsia monitoring)
- Weight
- Urine dipstick (protein -- preeclampsia screening)
- Fundal height (discordance >= 3 cm from expected triggers growth ultrasound)
- Fetal heart rate
- Fetal presentation assessment (Leopold maneuvers from 36 weeks)
- Fetal movement monitoring (kick counts from 28 weeks)

### Third Trimester Labs and Screenings

| Test | Gestational Age | LOINC | Notes |
|------|----------------|-------|-------|
| GBS culture | 36-37 weeks | 76685-3 | Vaginal/rectal swab |
| Repeat CBC | 36 weeks | 58410-2 | Assess for anemia before delivery |
| Repeat HIV | 36 weeks | 7917-8 | If high-risk |
| Repeat RPR/VDRL | 36 weeks | 20507-0 | If high-risk or per state law |
| Repeat GC/CT | 36 weeks | 43304-5, 43305-2 | If high-risk or positive earlier |
| Hepatitis B repeat | 36 weeks | 5196-1 | If high-risk |

### GBS Management

| GBS Status | Intrapartum Action |
|------------|-------------------|
| Positive | IV penicillin G (or ampicillin) at labor onset or ROM |
| Negative (screened 36-37 weeks) | No prophylaxis needed |
| Unknown at delivery | Treat if: preterm (<37 weeks), ROM >18 hours, fever >=100.4F, prior infant with GBS disease, GBS bacteriuria this pregnancy |
| PCN allergy (low risk) | Cefazolin |
| PCN allergy (high risk) | Clindamycin (if susceptible) or vancomycin |

### Fetal Surveillance Indications (Third Trimester)

Non-stress test (NST) and/or biophysical profile (BPP) indicated for:
- Gestational diabetes (on insulin): start 32 weeks, twice weekly
- Gestational hypertension / preeclampsia: start at diagnosis
- Decreased fetal movement
- Post-dates (>= 41 weeks)
- IUGR (intrauterine growth restriction)
- Oligohydramnios or polyhydramnios
- Multiple gestation
- Prior stillbirth

## Prenatal Risk Assessment Criteria

### Criteria for High-Risk Pregnancy Classification

**Maternal factors:**
- Age >= 35 (advanced maternal age) or < 17
- BMI >= 30 (obesity)
- Chronic hypertension
- Pre-existing diabetes (type 1 or 2)
- Autoimmune disease (SLE, antiphospholipid syndrome)
- Thrombophilia
- Chronic kidney disease
- Cardiac disease
- HIV positive
- Substance use disorder

**Obstetric history factors:**
- Prior preterm birth
- Prior preeclampsia
- Prior stillbirth or neonatal death
- Prior cesarean delivery
- Prior cervical insufficiency
- Recurrent pregnancy loss (>= 3)
- Prior gestational diabetes
- Uterine anomaly

**Current pregnancy factors:**
- Multiple gestation
- Placenta previa or accreta spectrum
- Short cervix (< 25mm before 24 weeks)
- Fetal anomaly
- IUGR
- Polyhydramnios / oligohydramnios
- Rh isoimmunization
- Gestational diabetes
- Preeclampsia

## Weight Gain Recommendations (IOM 2009)

| Prepregnancy BMI | BMI Range | Total Weight Gain | Weekly Gain (2nd/3rd Trimester) |
|-----------------|-----------|-------------------|-------------------------------|
| Underweight | < 18.5 | 28-40 lbs (12.5-18 kg) | ~1 lb/week (0.44-0.58 kg/wk) |
| Normal weight | 18.5-24.9 | 25-35 lbs (11.5-16 kg) | ~1 lb/week (0.35-0.50 kg/wk) |
| Overweight | 25.0-29.9 | 15-25 lbs (7-11.5 kg) | ~0.6 lb/week (0.23-0.33 kg/wk) |
| Obese (all classes) | >= 30.0 | 11-20 lbs (5-9 kg) | ~0.5 lb/week (0.17-0.27 kg/wk) |
| Twin pregnancy (normal BMI) | 18.5-24.9 | 37-54 lbs (17-25 kg) | -- |
| Twin pregnancy (obese) | >= 30.0 | 25-42 lbs (11-19 kg) | -- |

## Ultrasound Schedule

| Ultrasound Type | Gestational Age | Purpose |
|----------------|----------------|---------|
| Dating/viability | 6-9 weeks | Confirm intrauterine pregnancy, determine GA, detect multiple gestation |
| NT measurement | 11-14 weeks | Part of first trimester combined screen (optional) |
| Anatomy scan | 18-22 weeks | Comprehensive fetal anatomy survey, placental location, amniotic fluid, cervical length |
| Growth scan | 28-32 weeks | If indicated: IUGR concern, GDM, size/date discrepancy |
| Presentation | 36+ weeks | Confirm vertex presentation, BPP if indicated |
