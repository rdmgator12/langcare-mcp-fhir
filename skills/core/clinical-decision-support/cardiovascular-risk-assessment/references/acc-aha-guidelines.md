# ACC/AHA Guideline Recommendations Reference

## Statin Therapy Guidelines (2018 ACC/AHA Cholesterol Guideline)

### Four Statin Benefit Groups

| Group | Criteria | Recommendation |
|-------|----------|----------------|
| 1. Clinical ASCVD | Prior MI, ACS, stable angina, coronary/arterial revascularization, stroke, TIA, PAD | High-intensity statin (age <=75). Moderate-intensity if >75 or statin intolerant. |
| 2. LDL-C >= 190 mg/dL | Primary hypercholesterolemia (no secondary cause) | High-intensity statin. Consider ezetimibe or PCSK9 inhibitor if LDL-C remains >=100. |
| 3. Diabetes, age 40-75 | DM with LDL-C 70-189 mg/dL | Moderate-intensity statin. High-intensity if 10-year ASCVD risk >=20% or multiple risk enhancers. |
| 4. Primary prevention, age 40-75 | LDL-C 70-189 mg/dL, 10-year ASCVD risk >=7.5% | Moderate-to-high intensity statin based on risk discussion. |

### Statin Intensity

| Intensity | LDL-C Reduction | Medications |
|-----------|----------------|-------------|
| High | >= 50% | Atorvastatin 40-80mg, Rosuvastatin 20-40mg |
| Moderate | 30-49% | Atorvastatin 10-20mg, Rosuvastatin 5-10mg, Simvastatin 20-40mg, Pravastatin 40-80mg |
| Low | < 30% | Simvastatin 10mg, Pravastatin 10-20mg, Lovastatin 20mg, Fluvastatin 20-40mg |

### LDL-C Thresholds for Additional Therapy

| Clinical Scenario | LDL-C Threshold | Additional Therapy |
|-------------------|----------------|--------------------|
| Clinical ASCVD, on max statin | >= 70 mg/dL | Add ezetimibe |
| Clinical ASCVD, very high risk, on statin + ezetimibe | >= 70 mg/dL | Add PCSK9 inhibitor |
| LDL >= 190, on max statin | >= 100 mg/dL | Add ezetimibe, then PCSK9 inhibitor |
| Diabetes, on statin | >= 70 mg/dL (high risk) | Add ezetimibe |

### Very High-Risk ASCVD Criteria (for more aggressive LDL lowering)
- Multiple major ASCVD events (>=2 of: recent ACS within 12 months, MI other than above, ischemic stroke, symptomatic PAD)
- 1 major ASCVD event + multiple high-risk conditions (age >=65, heterozygous FH, prior CABG/PCI, DM, HTN, CKD stage 3+, current smoking, LDL >=100 on max statin, CHF)

## Anticoagulation in Atrial Fibrillation (2019 AHA/ACC/HRS Focused Update)

### Decision Algorithm Based on CHA2DS2-VASc

| CHA2DS2-VASc | Men | Women | Recommendation |
|-------------|-----|-------|----------------|
| 0 | 0 | 1 (sex alone) | No antithrombotic therapy (Class III - no benefit) |
| 1 | 1 | 2 | OAC may be considered (Class IIb). Shared decision-making. |
| >= 2 | >= 2 | >= 3 | OAC recommended (Class I). |

### Anticoagulant Selection

| Agent | Dose | Renal Adjustment | Key Considerations |
|-------|------|-------------------|-------------------|
| Apixaban | 5mg BID | 2.5mg BID if >=2 of: age >=80, weight <=60kg, Cr >=1.5 | Preferred in elderly, renal impairment. Lowest GI bleeding risk among DOACs. |
| Rivaroxaban | 20mg daily with food | 15mg daily if CrCl 15-50 mL/min | Once daily dosing. Take with evening meal. |
| Dabigatran | 150mg BID | 75mg BID if CrCl 15-30 mL/min | Only DOAC with reversal agent (idarucizumab). Avoid if CrCl <15. |
| Edoxaban | 60mg daily | 30mg daily if CrCl 15-50, weight <=60kg, or on P-gp inhibitors | Do not use if CrCl >95 mL/min (reduced efficacy). |
| Warfarin | Titrate to INR 2-3 | No renal adjustment | Preferred if mechanical valve or moderate-severe mitral stenosis. Requires INR monitoring. |

### DOAC vs Warfarin Selection

**Prefer DOACs when:**
- New initiation of anticoagulation
- Difficulty maintaining INR 2-3 (TTR <65%)
- Patient preference for no monitoring
- High intracranial hemorrhage risk

**Prefer Warfarin when:**
- Mechanical prosthetic heart valve
- Moderate-to-severe mitral stenosis
- Severe renal impairment (CrCl <15 mL/min for most DOACs)
- Cost/access barriers to DOACs
- Strong drug interactions with DOACs (strong dual P-gp and CYP3A4 inhibitors)

## Acute Chest Pain Workup by HEART Score Risk Level

### HEART 0-3 (Low Risk)

- 6-week MACE risk: 0.9-1.7%
- Disposition: Discharge home
- Required before discharge: single troponin (if 0-1h rule-out protocol) or serial troponins at 0 and 3 hours (if standard protocol)
- Follow-up: PCP within 72 hours, cardiology referral if recurrent symptoms
- Discharge instructions: Return for recurrent chest pain, dyspnea, syncope

### HEART 4-6 (Intermediate Risk)

- 6-week MACE risk: 12-16.6%
- Disposition: Observation unit or telemetry admission
- Workup:
  - Serial troponins at 0, 3, and 6 hours
  - Continuous telemetry monitoring
  - Repeat ECG if symptoms recur
  - Non-invasive testing within 24-72 hours:
    - **Stress test** (exercise or pharmacologic) if no ECG changes and troponins negative
    - **CCTA (coronary CT angiography)** if low-intermediate pretest probability
    - **Stress echocardiography** if structural assessment also needed
- If serial troponins negative and non-invasive test negative: discharge with cardiology follow-up
- If troponin rises or non-invasive test positive: upgrade to high-risk pathway

### HEART 7-10 (High Risk)

- 6-week MACE risk: 50-65%
- Disposition: Admit to telemetry or CCU
- Workup:
  - Serial troponins at 0, 3, and 6 hours
  - Cardiology consult (urgent)
  - Echocardiography (assess wall motion, EF)
  - Consider early invasive strategy (cardiac catheterization within 24 hours)
- Medical therapy:
  - Dual antiplatelet therapy (aspirin + P2Y12 inhibitor) if ACS confirmed
  - Anticoagulation (heparin)
  - Beta-blocker (if no contraindications)
  - Nitrates for ongoing ischemia
  - High-intensity statin

## Blood Pressure Treatment Targets

### 2017 ACC/AHA Hypertension Guideline

| Category | SBP (mmHg) | DBP (mmHg) | Classification |
|----------|------------|------------|----------------|
| Normal | < 120 | < 80 | No treatment |
| Elevated | 120-129 | < 80 | Lifestyle modifications |
| Stage 1 HTN | 130-139 | 80-89 | Lifestyle +/- medication if 10-year ASCVD risk >=10% |
| Stage 2 HTN | >= 140 | >= 90 | Lifestyle + medication (2 agents if SBP >=160 or DBP >=100) |

### Treatment Targets

| Population | SBP Target | Notes |
|------------|------------|-------|
| General, high CV risk | < 130/80 | SPRINT trial-based |
| Elderly >=65, ambulatory | < 130 systolic | If tolerated |
| Diabetes | < 130/80 | Per ADA 2023 |
| CKD | < 130/80 | Slows progression |
| Post-stroke | < 130/80 | Secondary prevention |

### First-Line Antihypertensive Classes

| Class | Examples | Preferred When |
|-------|----------|---------------|
| ACE inhibitors | Lisinopril, enalapril, ramipril | DM, CKD, HF (reduced EF), post-MI |
| ARBs | Losartan, valsartan, olmesartan | ACE intolerant (cough), same indications as ACE |
| CCBs (dihydropyridine) | Amlodipine, nifedipine | Elderly, African American, isolated systolic HTN |
| Thiazide diuretics | Chlorthalidone, hydrochlorothiazide, indapamide | Elderly, African American, osteoporosis |

### Compelling Indications

| Condition | Preferred Agents |
|-----------|-----------------|
| Heart failure (reduced EF) | ACEi/ARB + beta-blocker + diuretic + MRA |
| Post-MI | Beta-blocker + ACEi/ARB |
| Diabetes with proteinuria | ACEi or ARB |
| CKD | ACEi or ARB |
| Recurrent stroke prevention | ACEi + thiazide |
| Coronary artery disease | Beta-blocker + ACEi/ARB |
