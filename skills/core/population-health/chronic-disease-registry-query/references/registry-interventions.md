# Registry Intervention Triggers and Treatment Benchmarks

Intervention criteria by disease and severity, treatment pattern benchmarks, and outcome targets for chronic disease registry management.

---

## Diabetes Registry Interventions

### Therapy Escalation Triggers

| Trigger | Criteria | Recommended Action |
|---------|----------|--------------------|
| A1c not at goal after 3 months | A1c >= 7.0% on 2+ consecutive readings 3+ months apart | Add/intensify second agent |
| A1c severely elevated | A1c >= 9.0% | Consider insulin or GLP-1 agonist; endocrinology referral if A1c >= 10% |
| A1c worsening trend | A1c increased >= 0.5% over 6 months | Review adherence, add/change agent |
| Hypoglycemia events | >= 2 documented hypoglycemic episodes | Reduce sulfonylurea or insulin dose; consider switch to agent with lower hypo risk |
| Renal decline | eGFR < 60 or declining > 5/year | Initiate SGLT2 inhibitor (if not on one); adjust metformin if eGFR < 30 |
| New albuminuria | UACR > 30 mg/g (new finding) | Start ACEi/ARB if not on one; add SGLT2 inhibitor |
| New ASCVD event | MI, stroke, PAD diagnosis | High-intensity statin; consider GLP-1 agonist for cardiorenal benefit |

### Treatment Pattern Benchmarks

| Metric | Target | Rationale |
|--------|--------|-----------|
| % on metformin (first-line) | >= 85% | ADA first-line unless contraindicated |
| % on SGLT2i or GLP-1a with ASCVD/CKD | >= 70% | Cardiorenal benefit proven |
| % on statin (ages 40-75) | >= 80% | ADA/ACC recommendation |
| % on ACEi/ARB with albuminuria | >= 90% | KDIGO/ADA guideline |
| % with annual eye exam | >= 65% | HEDIS benchmark |
| % with annual foot exam | >= 75% | ADA standard of care |
| % with annual UACR | >= 70% | KDIGO/ADA guideline |

## Hypertension Registry Interventions

### Therapy Escalation Triggers

| Trigger | Criteria | Recommended Action |
|---------|----------|--------------------|
| BP not at goal on monotherapy | BP >= 130/80 after 1 month at max dose | Add second agent from different class |
| BP not at goal on dual therapy | BP >= 130/80 after 1 month | Add third agent (consider thiazide if not included) |
| Resistant hypertension | BP >= 130/80 on 3 agents (including diuretic) | Evaluate secondary causes; MRA; specialist referral |
| Hypertensive urgency | BP >= 180/120 without end-organ damage | Same-day medication adjustment; 1-week recheck |
| Hypertensive emergency | BP >= 180/120 with symptoms/end-organ damage | ED referral |
| New target organ damage | New LVH, CKD, retinopathy, proteinuria | Intensify therapy; specialist referral |
| White coat hypertension suspected | Office BP elevated, home BP normal | ABPM or home BP monitoring program |

### Treatment Pattern Benchmarks

| Metric | Target | Rationale |
|--------|--------|-----------|
| % on any antihypertensive | >= 95% | All diagnosed patients should be on therapy |
| % on ACEi/ARB (with DM or CKD) | >= 80% | First-line for cardiorenal protection |
| % on thiazide (without CKD) | >= 40% | Evidence-based first-line agent |
| % with home BP monitoring | >= 30% | Improves control, identifies white coat HTN |
| % with annual metabolic panel | >= 85% | Monitor electrolytes on therapy |
| Mean number of antihypertensives | 1.5-2.5 | Most patients need 2+ agents |

## CHF Registry Interventions

### GDMT Optimization Triggers

| Trigger | Criteria | Recommended Action |
|---------|----------|--------------------|
| Not on ACEi/ARB/ARNI | HFrEF and no contraindication | Start ACEi; switch to ARNI if tolerated |
| Not on beta blocker | HFrEF and no contraindication | Start carvedilol, metoprolol succinate, or bisoprolol |
| Not on MRA | HFrEF, NYHA II-IV, and eGFR > 30, K < 5.0 | Start spironolactone or eplerenone |
| Not on SGLT2i | HFrEF or HFpEF | Start dapagliflozin or empagliflozin |
| BNP rising | BNP > 2x prior stable value | Evaluate volume status; diuretic adjustment; consider hospitalization if BNP > 500 |
| Weight gain > 3 lb/day | Daily weight monitoring | Increase diuretic; same-day evaluation if > 5 lb/week |
| NYHA worsening | Class change (e.g., II to III) | Optimize all GDMT pillars; cardiology referral |
| LVEF worsening | Decline > 5% from prior echo | Reassess therapy; advanced HF evaluation if LVEF < 25% |
| Recurrent hospitalization | >= 2 CHF hospitalizations in 12 months | Advanced HF referral; consider device therapy (ICD/CRT) |

## COPD Registry Interventions

### Therapy Escalation Triggers

| Trigger | Criteria | Recommended Action |
|---------|----------|--------------------|
| >= 2 moderate exacerbations/year | Exacerbation count from Condition/Encounter | Escalate to ICS/LABA/LAMA triple therapy |
| >= 1 hospitalization for exacerbation | Encounter class=IMP with COPD exacerbation | Add ICS if not on one; pulmonology referral |
| FEV1 declining > 60 mL/year | Serial spirometry trend | Reassess therapy; occupational/environmental exposure review |
| SpO2 < 88% at rest | Observation code 2708-6 | Evaluate for supplemental oxygen |
| Persistent symptoms despite dual therapy | CAT >= 10 or mMRC >= 2 on LABA/LAMA | Add ICS (if blood eosinophils >= 300) |
| Active smoker | LOINC 72166-2 = current smoker | Smoking cessation intervention at every visit |

## CKD Registry Interventions

### Therapy Triggers

| Trigger | Criteria | Recommended Action |
|---------|----------|--------------------|
| eGFR decline > 5 mL/min/year | Serial eGFR trend | Nephrology referral; review nephrotoxic meds |
| New proteinuria or worsening | UACR > 30 (new) or > 300 (worsening) | Start/maximize ACEi/ARB; add SGLT2i |
| eGFR < 30 | Single value < 30 | Nephrology referral; prepare for RRT education |
| eGFR < 15 | Single value < 15 | Urgent nephrology; dialysis access planning |
| Hemoglobin < 10 | LOINC 718-7 | Evaluate iron studies; consider ESA; nephrology input |
| Potassium > 5.5 | LOINC 2823-3 | Dietary counseling; review ACEi/ARB/MRA doses; consider potassium binder |
| Phosphorus > 4.5 (CKD 3-5) | LOINC 2777-1 | Phosphate binder; dietary restriction |
| Metabolic acidosis | Bicarb < 22, LOINC 1963-8 | Oral sodium bicarbonate supplementation |
| BP > 130/80 | LOINC 85354-9 | Optimize antihypertensives (ACEi/ARB preferred) |

### Outcome Targets per CKD Stage

| Metric | CKD 1-2 | CKD 3 | CKD 4 | CKD 5 |
|--------|---------|-------|-------|-------|
| eGFR decline | < 3 mL/min/yr | < 3 mL/min/yr | < 5 mL/min/yr | Dialysis planning |
| UACR | < 30 mg/g | < 300 mg/g | Minimize | -- |
| BP | < 130/80 | < 130/80 | < 130/80 | < 140/90 (on dialysis) |
| Hemoglobin | Normal | >= 10 | >= 10 | 10-11.5 |
| Potassium | 3.5-5.0 | 3.5-5.0 | 3.5-5.0 | 3.5-5.5 |
| Phosphorus | Normal | < 4.5 | < 4.5 | 3.5-5.5 |

## General Registry Management Cadence

### Recommended Review Frequency

| Risk Level | Registry Review | Patient Contact | Lab Frequency |
|------------|----------------|-----------------|---------------|
| High (out of control, complications, hospitalized) | Monthly | Weekly-biweekly | Per protocol + PRN |
| Medium (above target, stable) | Quarterly | Monthly | Per protocol |
| Low (at target, stable) | Biannually | Quarterly | Per protocol |

### Registry Report Cycle

- **Weekly**: High-risk patient list for care management review
- **Monthly**: Aggregate metrics, new enrollments, hospital readmissions
- **Quarterly**: Full registry report with trend analysis, intervention trigger review
- **Annually**: Year-over-year comparison, outcome analysis, quality measure alignment
