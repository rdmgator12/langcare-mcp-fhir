# TNM Staging and RECIST Response Criteria Reference

## AJCC TNM Staging (8th Edition -- General Framework)

### T -- Primary Tumor
| Stage | Definition |
|-------|-----------|
| TX | Tumor cannot be assessed |
| T0 | No evidence of primary tumor |
| Tis | Carcinoma in situ |
| T1 | Small tumor, limited to organ of origin |
| T2 | Larger tumor or minor local extension |
| T3 | Tumor extends beyond organ but not to adjacent structures |
| T4 | Tumor invades adjacent structures |

### N -- Regional Lymph Nodes
| Stage | Definition |
|-------|-----------|
| NX | Nodes cannot be assessed |
| N0 | No regional node metastasis |
| N1 | Limited regional node involvement |
| N2 | More extensive regional node involvement |
| N3 | Extensive regional node involvement |

### M -- Distant Metastasis
| Stage | Definition |
|-------|-----------|
| M0 | No distant metastasis |
| M1 | Distant metastasis present |
| M1a | Single distant organ metastasis |
| M1b | Multiple organ metastases |

### Overall Stage Grouping (General Pattern)
| Stage | Typical TNM | 5-Year Survival (varies by cancer) |
|-------|------------|-----------------------------------|
| Stage 0 | Tis N0 M0 | >95% |
| Stage I | T1-2 N0 M0 | 70-90% |
| Stage II | T2-3 N0-1 M0 | 50-70% |
| Stage III | T3-4 N1-2 M0 | 25-50% |
| Stage IV | Any T, Any N, M1 | 5-20% |

Note: Staging criteria are cancer-type specific. These are general patterns only.

## RECIST 1.1 (Response Evaluation Criteria in Solid Tumors)

### Target Lesion Selection
- Maximum 5 lesions total (2 per organ)
- Measurable lesions >= 10mm in longest diameter (>= 15mm for lymph nodes short axis)
- Sum of longest diameters (SLD) at baseline

### Response Categories

| Response | Criteria | Abbreviation |
|----------|----------|--------------|
| Complete Response | Disappearance of all target lesions; any pathological lymph nodes reduced to <10mm short axis | CR |
| Partial Response | >= 30% decrease in SLD from baseline | PR |
| Stable Disease | Neither sufficient shrinkage for PR nor sufficient increase for PD | SD |
| Progressive Disease | >= 20% increase in SLD from nadir (and >= 5mm absolute increase) OR new lesion(s) | PD |

### Overall Response
| Best Overall Response | Requires |
|----------------------|----------|
| CR | CR + CR (target + non-target) + no new lesions |
| PR | CR or PR + non-CR/non-PD (non-target) + no new lesions |
| SD | SD + non-CR/non-PD + no new lesions |
| PD | PD in any category OR new lesion |

### Confirmation Requirements
- CR requires confirmation >= 4 weeks later
- PR requires confirmation >= 4 weeks later (in clinical trials)

## ECOG Performance Status

| Score | Description |
|-------|-----------|
| 0 | Fully active, no restrictions |
| 1 | Restricted in strenuous activity; able to carry out light work |
| 2 | Ambulatory and capable of self-care; unable to carry out work; up and about >50% of waking hours |
| 3 | Capable of only limited self-care; confined to bed/chair >50% of waking hours |
| 4 | Completely disabled; cannot carry out any self-care; totally confined to bed/chair |
| 5 | Dead |

ECOG 0-1: eligible for most clinical trials and aggressive therapy
ECOG 2: consider treatment benefit/risk carefully
ECOG 3-4: generally supportive/palliative care only

## Chemotherapy Lab Monitoring

### CBC Monitoring
| Phase | Timing | Key Values |
|-------|--------|-----------|
| Pre-cycle | Day 1 of each cycle | ANC >= 1500, Plt >= 100K typically required to treat |
| Nadir | Day 7-14 (varies by regimen) | ANC < 500 = severe neutropenia; Plt < 20K = transfusion threshold |
| Recovery | Day 21-28 | Should recover before next cycle |

### Febrile Neutropenia Definition
- ANC < 500/uL (or expected to decline to <500) AND
- Single oral temperature >= 38.3C (101F) or >= 38.0C (100.4F) sustained > 1 hour
- ONCOLOGIC EMERGENCY -- requires immediate blood cultures and empiric antibiotics

### Tumor Lysis Syndrome Labs
| Lab | Abnormality | Action |
|-----|------------|--------|
| Potassium | > 6.0 mEq/L | Urgent management (ECG, insulin/glucose, kayexalate) |
| Uric acid | > 8.0 mg/dL | Rasburicase or allopurinol |
| Phosphate | > 4.5 mg/dL | Phosphate binders |
| Calcium | < 7.0 mg/dL (due to phosphate) | Treat underlying hyperphosphatemia |
| LDH | Elevated | Marker of cell lysis (not directly treated) |
| Creatinine | Rising | Aggressive hydration, rasburicase; consider dialysis |
