# ACOG Prenatal Visit and Lab Schedule Reference

## Visit Frequency

| Gestational Age | Visit Frequency |
|----------------|-----------------|
| 4-28 weeks | Every 4 weeks |
| 28-36 weeks | Every 2 weeks |
| 36 weeks - delivery | Weekly |

## First Trimester (Initial Visit / 0-13 weeks)

### Labs
| Test | LOINC | Purpose |
|------|-------|---------|
| Blood type + Rh | 883-9, 10331-7 | ABO/Rh typing, Rh immunoprophylaxis planning |
| Antibody screen | 890-4 | Alloantibody detection |
| CBC with differential | 58410-2 | Anemia screening, baseline |
| Rubella IgG | 5334-8 | Immunity verification |
| Hepatitis B surface antigen | 5196-1 | HBV carrier status |
| HIV screening | 7917-8 | Universal screening |
| RPR/VDRL (syphilis) | 20507-0 | Universal screening |
| Urine culture | 630-4 | Asymptomatic bacteriuria (treat if positive) |
| Chlamydia NAAT | 43304-5 | Universal <25 or risk factors |
| Gonorrhea NAAT | 43305-2 | Universal <25 or risk factors |
| Pap smear | 10524-7 | If due per cervical cancer screening schedule |
| Urine protein | 5804-0 | Baseline for preeclampsia monitoring |
| TSH | 3016-3 | If symptomatic or high-risk |

### Procedures
| Procedure | Timing | Purpose |
|-----------|--------|---------|
| Dating ultrasound | 6-9 weeks | Confirm viability, establish EDD |
| NT scan + first trimester screen | 11-14 weeks | Aneuploidy risk (optional) |
| Cell-free DNA (cfDNA) | 10+ weeks | Aneuploidy screening (optional) |

## Second Trimester (14-27 weeks)

### Labs (24-28 weeks)
| Test | LOINC | Purpose |
|------|-------|---------|
| 1-hour glucose challenge | 1504-0 | GDM screening (positive if >=130 or >=140 per institution) |
| 3-hour GTT (if GCT positive) | 1518-0, 1530-5, 1547-9, 1507-3 | GDM diagnosis (Carpenter-Coustan: fasting >=95, 1h >=180, 2h >=155, 3h >=140) |
| Repeat Rh antibody screen | 890-4 | If Rh-negative |
| CBC | 58410-2 | Anemia reassessment |

### Procedures
| Procedure | Timing | Purpose |
|-----------|--------|---------|
| Anatomy scan | 18-22 weeks | Fetal anatomy survey, placental location |
| Rhogam (if Rh-negative) | 28 weeks | Rh immunoprophylaxis |
| Quad screen (if no cfDNA) | 15-20 weeks | Aneuploidy + NTD screening |

## Third Trimester (28-40+ weeks)

### Labs (35-37 weeks)
| Test | LOINC | Purpose |
|------|-------|---------|
| GBS culture (vaginal-rectal) | 76685-3 | Intrapartum prophylaxis decision |
| Repeat CBC | 58410-2 | Anemia check before delivery |
| Repeat HIV | 7917-8 | High-risk only (or state mandate) |
| Repeat syphilis (RPR) | 20507-0 | High-risk or state mandate |
| Repeat chlamydia/gonorrhea | 43304-5, 43305-2 | High-risk only |

### Monitoring
| Assessment | Timing | Purpose |
|-----------|--------|---------|
| Fetal kick counts | From 28 weeks daily | Fetal well-being |
| NST (non-stress test) | Weekly from 32-34 weeks (high-risk) | Fetal heart rate reactivity |
| BPP (biophysical profile) | As indicated | Comprehensive fetal assessment |
| Growth ultrasound | 28-32 weeks (if indicated) | EFW, amniotic fluid |

## Risk-Based Additional Monitoring

### Preeclampsia High-Risk Factors (Initiate aspirin 81mg at 12-28 weeks)
- Prior preeclampsia
- Multifetal gestation
- Chronic hypertension
- Type 1 or Type 2 diabetes
- Renal disease
- Autoimmune disease (SLE, antiphospholipid syndrome)

### Preeclampsia Moderate-Risk Factors (Aspirin if multiple present)
- Nulliparity
- Obesity (BMI >30)
- Family history of preeclampsia
- Age >=35
- Prior adverse pregnancy outcome
- IVF pregnancy
- >10 years since last pregnancy

### GDM Criteria (Carpenter-Coustan)
| Timepoint | Threshold (mg/dL) |
|-----------|-------------------|
| Fasting | >= 95 |
| 1-hour | >= 180 |
| 2-hour | >= 155 |
| 3-hour | >= 140 |

Diagnosis requires >= 2 abnormal values on 3-hour GTT.
