# Morse Fall Scale Reference

## Morse Fall Scale Scoring

| Factor | Criteria | Score |
|--------|----------|-------|
| History of falling (within 3 months) | No | 0 |
| | Yes | 25 |
| Secondary diagnosis | No | 0 |
| | Yes (>=2 medical diagnoses) | 15 |
| Ambulatory aid | None / bed rest / nurse assist | 0 |
| | Crutches / cane / walker | 15 |
| | Furniture (walks holding onto furniture) | 30 |
| IV/heparin lock | No | 0 |
| | Yes | 20 |
| Gait/transferring | Normal / bed rest / immobile | 0 |
| | Weak | 10 |
| | Impaired (shuffling, short steps, stumbles) | 20 |
| Mental status | Oriented to own ability | 0 |
| | Overestimates ability / forgets limitations | 15 |

**Maximum Score: 125**

## Risk Classification and Interventions

| Score Range | Risk Level | Standard Interventions |
|-------------|-----------|----------------------|
| 0-24 | Low Risk | Good basic nursing care: clean environment, call light in reach, non-skid footwear, bed in low position |
| 25-50 | Moderate Risk | All low-risk interventions PLUS: fall risk signage, scheduled toileting, assist with ambulation, remove unnecessary clutter, nightlight |
| > 50 | High Risk | All moderate-risk interventions PLUS: 1:1 or close monitoring, bed alarm, chair alarm, physical therapy consult, consider sitter, non-pharmacologic sleep aids, medication review |

## Fall-Risk Medications

### High-Risk Classes (Strong Evidence)

| Drug Class | Examples | Mechanism |
|-----------|----------|-----------|
| Benzodiazepines | Alprazolam, lorazepam, diazepam, clonazepam | Sedation, impaired balance, psychomotor impairment |
| Non-benzo hypnotics (Z-drugs) | Zolpidem, zaleplon, eszopiclone | Sedation, next-day impairment |
| Opioids | All | Sedation, dizziness, orthostatic hypotension |
| Antipsychotics | Haloperidol, quetiapine, risperidone, olanzapine | Sedation, orthostatic hypotension, EPS |
| Tricyclic antidepressants | Amitriptyline, nortriptyline, doxepin | Anticholinergic effects, sedation, orthostatic hypotension |
| First-gen antihistamines | Diphenhydramine, hydroxyzine, promethazine | Sedation, anticholinergic effects |

### Moderate-Risk Classes

| Drug Class | Examples | Mechanism |
|-----------|----------|-----------|
| SSRIs/SNRIs | Sertraline, citalopram, venlafaxine, duloxetine | Hyponatremia, dizziness |
| Anticonvulsants | Gabapentin, pregabalin, phenytoin, carbamazepine | Sedation, dizziness, ataxia |
| Alpha-1 blockers | Doxazosin, prazosin, tamsulosin, terazosin | Orthostatic hypotension |
| Centrally-acting antihypertensives | Clonidine, methyldopa | Sedation, orthostatic hypotension |
| Diuretics | Furosemide, HCTZ | Volume depletion, orthostatic hypotension, electrolyte imbalance |
| Muscle relaxants | Cyclobenzaprine, tizanidine, methocarbamol | Sedation, dizziness |
| Anticholinergics | Oxybutynin, tolterodine, solifenacin | Confusion, blurred vision |

## CDC STEADI Algorithm

1. **Screen**: Ask 3 key questions annually for patients >= 65:
   - Have you fallen in the past year?
   - Do you feel unsteady when standing or walking?
   - Do you worry about falling?
2. **Assess**: If any "yes":
   - Timed Up and Go (TUG) test: >= 12 seconds = increased risk
   - 30-Second Chair Stand: below age-adjusted threshold = weakness
   - 4-Stage Balance Test: unable to hold tandem for 10 seconds = balance impairment
3. **Intervene**:
   - Medication review (reduce/eliminate fall-risk medications)
   - Vitamin D supplementation (800-1000 IU/day if deficient)
   - Vision check and correction
   - Home hazard assessment
   - Exercise program (balance + strength training)
   - Address orthostatic hypotension
   - Manage foot problems and footwear

## Orthostatic Hypotension Assessment

**Positive if:**
- Systolic BP drop >= 20 mmHg within 3 minutes of standing
- Diastolic BP drop >= 10 mmHg within 3 minutes of standing
- Heart rate increase > 30 bpm (suggests volume depletion)

**Protocol:**
1. Supine BP/HR after 5 minutes rest
2. Standing BP/HR at 1 minute
3. Standing BP/HR at 3 minutes
4. Document symptomatic (dizziness, lightheadedness) vs asymptomatic
